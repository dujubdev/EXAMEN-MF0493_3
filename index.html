Bandas de Rock Legendarias

Sobre el Proyecto

Rockify es una aplicación web progresiva (PWA) dedicada a las bandas de rock más legendarias de los 70s, 80s y 90s. Desarrollada con HTML5, CSS3 y JavaScript vanilla.

Tecnologías

  • HTML5 Semántico
  • CSS3 Responsivo
  • JavaScript ES6+
  • Service Worker
  • LocalStorage

APIs Utilizadas

  • YouTube Embed
  • TheAudioDB
  • OpenWeatherMap
